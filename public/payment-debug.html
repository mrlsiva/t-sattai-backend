🔍 Backend Requirements Analysis

Payment Intent Creation Validation:

  • amount: required, numeric, min 0.50
  • currency: optional, string, must be usd/eur/gbp
  • Cart verification: Backend checks if cart matches amount
  • Stock validation: Ensures all cart items have sufficient stock

Payment Confirmation Validation:

  • payment_intent_id: required string
  • shipping_address: required object with specific fields
  • shipping_address.name: required string
  • shipping_address.line1: required string
  • shipping_address.line2: optional string
  • shipping_address.city: required string
  • shipping_address.state: optional string
  • shipping_address.postal_code: required string
  • shipping_address.country: required string, exactly 2 characters

Backend Business Logic:

  • Verifies payment intent status is 'succeeded' in Stripe
  • Checks payment belongs to authenticated user
  • Prevents duplicate orders for same payment intent
  • Calculates totals: subtotal + tax (8%) + shipping ($10)
  • Creates order and order items from cart
  • Updates product stock levels
  • Clears user's cart after successful order
